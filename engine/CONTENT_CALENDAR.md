# Re-Self 90-Day Content Calendar

---

## How to Use This Calendar

This document provides a detailed, day-by-day posting schedule for the entire 90-day Re-Self marketing campaign. It integrates both the **Main Content Arc** and the adjacent **"Straight Shot" Mini-Course promotions** into a single, unified timeline.

- **Timezone:** All times are suggestions. Please adjust to your primary target audience's local timezone (e.g., US Pacific Time).
- **Post Content:** The full copy for each post can be found in the `EXECUTION_GUIDE.md` document.
- **Image Assets:** The specific image for each post is listed in the `IMAGE_ASSIGNMENT_GUIDE.md` and the `EXECUTION_GUIDE.md`.
