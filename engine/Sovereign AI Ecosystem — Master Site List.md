## DNS Configuration Quick Reference

For all domains, the DNS records to point to Vercel are:

| Record Type | Host | Value |
| :--- | :--- | :--- |
| A Record | `@` | `76.76.21.21` |
| CNAME Record | `www` | `cname.vercel-dns.com` |

Detailed step-by-step instructions for configuring DNS are available in the separate GoDaddy and Namecheap instruction documents.
