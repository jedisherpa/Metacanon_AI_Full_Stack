## Part 5: The New, Easier Way to Create Project Folders

This workflow is now much simpler. The new Orchestrator Prompt is designed to **automatically detect and use agent files** within a project folder.

Here’s how to set up a reusable project for your **Marketing Branding Board**.

### Step 1: Create the Project Directory

> "Create a new directory for my marketing projects at `/home/ubuntu/projects/MarketingBrandCouncil`."

### Step 2: Add Your Team and Rules

1.  **Copy the Agent Personas:**
    > "Copy all the agent files from `/home/ubuntu/marcus_manus_v2/Marcus Manus Files/Marketing Branding Board Agents/` into my new `/home/ubuntu/projects/MarketingBrandCouncil/` directory."

2.  **Copy the Governance Files:**
    > "Copy the `v3.0_Metacanon_Constitution.md` and `ORCHESTRATOR-PROMPT-SUBSPHERE-SYNTHESIS.docx` from `/home/ubuntu/constitution/` into `/home/ubuntu/projects/MarketingBrandCouncil/`."

### Step 3: Create the Automatic Activation Prompt (`MANUS_INSTRUCTIONS.md`)

This file now only needs to tell Manus its role. It will handle the rest automatically.

> "Create a new file at `/home/ubuntu/projects/MarketingBrandCouncil/MANUS_INSTRUCTIONS.md` with the following text:
> 
> # Marketing Council Activation Protocol
> 
> You are to act as the **Constitutional Orchestrator** for all tasks in this project. Your sole function is to facilitate the process according to the rules defined in the governance documents located in this project folder.
> 
> You are a non-voting, non-sovereign facilitator. You must not inject your own opinion or make any decisions. The Orchestrator will automatically detect and use the agent persona files present in this folder.
> 
> Unless otherwise specified, you should use the `strategic-brand-council` skill for tasks initiated in this project.

### Step 4: Use Your Project

Now, just go to the `MarketingBrandCouncil` project in Manus and start a new task. You can simply say:

> "I need to develop a brand strategy for a new SaaS product. Please begin the deliberation process."

M Manus will read the instructions, adopt the Orchestrator role, and automatically load the marketing experts you placed in the folder to begin the work. No long activation prompt needed.

## Part 6: The New Skill - Perspectival Agent Synthesis

I've added a powerful new skill to your toolkit: **Perspectival Agent Synthesis**.

This skill is for **creating new AI agent personas from scratch**. If you have a collection of writings, transcripts, or other documents from a person, this skill can analyze them and generate a high-fidelity agent prompt that simulates their unique way of thinking.

### How to Use It

1.  Create a new project folder.
2.  Place all the text documents you have from the person you want to model into that folder.
3.  Start a new task in that folder and use the following prompt:

> "Using the `perspectival-agent-synthesis` skill, analyze all the text documents in this folder and generate a new agent persona prompt for me."

M Manus will then orchestrate a team of 6 specialized AI agents to analyze the texts from different angles (cognitive, linguistic, psychological, etc.) and synthesize a detailed agent persona file for you to use in your other projects.
